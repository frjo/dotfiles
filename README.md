dotfiles
========

My dot files for zsh, git, vim and tmux.

The following files and directories have had the leading dot in the name removed:

* aliases -> .aliases
* gitconfig -> .gitconfig
* gitignore -> .gitignore
* tmux.conf -> .tmux.conf
* vimrc -> .vimrc
* zsh/ -> .zsh/
* zsh/zshrc -> .zsh/.zshrc
* zshenv -> .zshenv
